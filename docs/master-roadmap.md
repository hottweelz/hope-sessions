# Master Roadmap

## Purpose

This is the integrated operating roadmap for the Denver, North Carolina grant effort, the Hope initiative, and the PMA web modernization track. The goal is to go all out without losing the original mission.

The original mission still comes first:

1. identify realistic funding targets
2. build the reusable grant package
3. submit at least one direct artist-level application

Everything else supports that.

## Strategic North Star

The project is not asking funders to subsidize a recording business. The project is building a Hope-centered creative initiative led by Paul Andrejack, using PMA Recording Studios as infrastructure for music creation, artist development, access, and future community-facing programming.

## Primary Workstreams

### Workstream 1: Grant Acquisition Core

This remains the primary workstream.

Primary outcomes:

- one verified opportunity map
- one canonical narrative bank
- one evidence kit
- one budget framework
- one submission tracker
- one direct artist-level application package

Primary target:

- North Carolina Artist Support Grants via Arts & Science Council Charlotte-Mecklenburg for Lincoln County

### Workstream 2: Hope Initiative Development

This workstream turns Paul Andrejack and Andrejack's album Hope into the central artistic and public-facing concept that unifies grant writing, outreach, and future programming.

Primary outcomes:

- one flagship Hope project definition
- one public-facing initiative narrative
- one partner-facing concept statement
- one scalable path from direct artist support to community programming

### Workstream 3: PMA Public Credibility Layer

This is a support workstream, not the primary objective.

Primary outcomes:

- one supplemental HTML5/CSS Hope Sessions page or microsite section
- one public proof surface for partners and outreach
- one roadmap for the broader 2026 PMA refresh

## Priority Order

### Tier 1: Must Happen First

1. verify the ASC Artist Support local cycle details
2. lock the first Hope project scope
3. assemble the evidence kit and work samples
4. draft the first application package

### Tier 2: Must Happen In Parallel If Capacity Allows

1. develop the Hope Sessions public narrative
2. identify partner prospects in Lincoln County and nearby regions
3. prepare the supplemental PMA page specification

### Tier 3: Valuable But Must Not Delay Submission

1. full PMA website redesign
2. broader content marketing rollout
3. long-horizon private and partner-funded expansion packaging

## 90-Day Integrated Plan

### Days 1-30

Grant core:

- confirm active ASC timing and local requirements
- select the exact first-cycle Hope project format
- assemble bio, residency proof, resume, work samples, and base budget inputs

Hope initiative:

- finalize the Hope Sessions description and messaging guardrails
- decide whether the first project is new work, session series, or artist-development focused

Web:

- define the supplemental page audience and content needs
- inventory proof assets already available from PMA and Andrejack

### Days 31-60

Grant core:

- finish the narrative bank and budget template
- complete the first application draft
- identify and contact at least three relevant partner prospects

Hope initiative:

- write the partner-facing concept statement
- translate the initiative into artist-facing and partner-facing language

Web:

- finalize one-page information architecture
- draft page copy and proof blocks
- define needed photography, media embeds, and calls to action

### Days 61-90

Grant core:

- harden the application package
- run eligibility, budget, and evidence review
- submit the direct artist-level application if the decision standard is met

Hope initiative:

- convert the application narrative into outreach materials
- prepare letters-of-support requests and partner outreach copy

Web:

- finalize the supplemental page build plan so it can launch without blocking the grant cycle

## 180-Day Expansion Plan

1. convert the first Hope project into documented proof of concept
2. establish at least three meaningful partner relationships
3. test Hope Sessions access, workshop, or story-based formats
4. identify a viable fiscal-agent or nonprofit partnership path for larger opportunities
5. launch the supplemental Hope page if not already live

## 12-Month Outcome Plan

1. maintain a repeatable annual grants calendar
2. sustain one to three funding wins or partnership-backed opportunities
3. turn PMA from a studio-only brand into a documented community arts asset with an artist-development and access story
4. complete the broader 2026 PMA public-site modernization

## Decision Rules

### Keep Doing

- anything that moves the direct application toward submission readiness
- anything that strengthens Hope as a real artistic and public-benefit frame
- anything that improves partner confidence with evidence

### Do Not Let Expand Unchecked

- web design work that delays the grant package
- branding ideas without proof or delivery logic
- community claims that outrun actual program capacity
- commercial studio messaging that muddies the fundable mission

## Final Operating Principle

The grant package is the engine. Hope is the narrative spine. The PMA web layer is the amplifier. If priorities conflict, protect the engine first.
