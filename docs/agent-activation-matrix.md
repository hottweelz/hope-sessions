# Agent Activation Matrix

## Purpose

This matrix shows which agents should be active for each major workstream so the team stays coordinated and relevant instead of noisy.

## Workstream 1: Direct Grant Submission

### Active Core

- Agents Orchestrator
- Senior Project Manager
- Project Shepherd
- Trend Researcher
- Proposal Strategist
- Finance Tracker
- Legal Compliance Checker
- Technical Writer
- Reality Checker
- Evidence Collector

### Advisory Bench

- Product Manager
- Brand Guardian
- Executive Summary Generator
- Workflow Optimizer

### Why

This mix keeps the team centered on verified opportunities, narrative quality, compliance, budget logic, and submission readiness.

## Activation Rule

If a workstream is not directly moving one of the original 90-day objectives, it should not displace the direct grant submission workstream from the top priority slot.
