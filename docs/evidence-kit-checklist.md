# Evidence Kit Checklist

This is the working evidence checklist for the first application cycle.

## Identity And Eligibility

- Applicant legal name confirmed
- Preferred applying posture confirmed: individual artist, business entity, or future fiscal-agent route
- Current mailing address confirmed
- Lincoln County residency proof collected
- Government ID available if portal requires identity verification

## Core Narrative Assets

- One-sentence positioning approved
- Short bio drafted
- Long bio drafted
- Artist resume or CV assembled
- Artist statement drafted
- Hope initiative description drafted
- Flagship project title chosen
- Flagship project description drafted
- Community impact statement drafted

## Work Samples

- Three to five strongest work samples shortlisted
- Each sample has title, date, role, and format listed
- Streaming or download links verified
- Hope album and related Andrejack materials assessed for strongest grant-relevant excerpts
- One sample set mapped to the exact ask in the proposal
- Work sample list document drafted if required

## Budget And Delivery Proof

- Project timeline drafted
- Lean budget drafted
- Standard budget drafted
- Budget notes explain each major cost in output terms
- Any equipment line is tied to delivery, access, or quality outcomes
- Feasibility notes drafted for project completion within the grant period

## Community And Partner Proof

- Initial partner target list built
- At least three likely partners identified
- Letter-of-support request draft created
- Any current collaborator commitments documented
- Public-benefit claims limited to what can actually be delivered

## Submission Assets

- Opportunity-specific checklist built for the target application
- Portal account or application gateway confirmed
- File naming convention decided
- Final PDF export workflow decided if needed

## Current Missing Items

- Active 2026 ASC local cycle confirmation
- Final Hope flagship project scope
- Final work sample selection
- Residency proof packet
- Budget inputs from actual production plan
- Partner list with priorities
