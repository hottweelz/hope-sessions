# Asset Usage Note

## Current Recommended Asset Hierarchy

### Primary Assets To Use

1. PMA logo
2. Hope album cover
3. Paul Andrejack portrait or PMA studio documentary photography when available

These assets are the safest and most strategically aligned for the first public Hope Sessions page.

## Two Sides To Every Story Image

The Two Sides To Every Story image is visually strong and editorially compelling. It could work as a secondary story image in the proof or narrative layer.

However, it should not be treated as automatically approved for a public website just because it is available. The image depicts a real identifiable child, and the user notes that the child is autistic. That introduces a higher standard for reuse.

### Practical Rule

Only use this image on the public Hope Sessions or PMA site if all of the following are true:

1. the rights holder approves website reuse
2. the parent or guardian has explicitly approved public website use
3. the team is comfortable with the image being used as a respectful editorial element rather than emotional leverage

## Best Current Recommendation

For the first website pass:

- use the PMA logo as the brand anchor
- use the Hope cover as the campaign anchor
- keep the Two Sides image off the public page until permission status is explicit

If permission is confirmed later, it is better used as a secondary narrative image than as the hero.
