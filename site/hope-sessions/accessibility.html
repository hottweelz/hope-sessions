Accessibility Statement

Last reviewed: April 13, 2026  ·  Standard: WCAG 2.2 Level AA

Our Commitment

PMA Recording Studios and Hope Sessions are committed to ensuring that this website is accessible to everyone, including people with disabilities. We strive to meet the Web Content Accessibility Guidelines (WCAG) 2.2, Level AA and to comply with the Americans with Disabilities Act (ADA), Title III.

We believe that accessibility is not a feature — it is a fundamental requirement. Hope Sessions is an initiative about removing barriers for artists. We hold ourselves to the same standard on the web.

What We Have Implemented

  • Keyboard navigation: All interactive elements are reachable and operable using a keyboard alone.
  • Skip link: A “Skip to main content” link is the first focusable element on each page.
  • Semantic HTML5: Proper landmark regions (<header>, <main>, <nav>, <footer>) are used throughout.
  • Heading hierarchy: A logical heading structure (h1 → h2 → h3) is maintained on all pages.
  • Color contrast: All text meets WCAG 2.2 AA minimum contrast ratios (4.5:1 for normal text, 3:1 for large text). Primary text achieves approximately 14:1 contrast against the dark background.
  • Focus indicators: Visible focus rings are present on all interactive elements and meet WCAG 2.4.11 Focus Appearance requirements.
  • Alternative text: All meaningful images have descriptive alt attributes. Decorative images use empty alt text.
  • Form accessibility: All form fields have explicit <label> elements, radio groups use <fieldset>/<legend>, and error messages are associated via aria-describedby.
  • ARIA: Landmark labels (aria-label, aria-labelledby) are applied to sections and navigation regions to aid screen reader navigation.
  • Motion: Animations and transitions are suppressed when the user has enabled prefers-reduced-motion.
  • Forced colors: The site respects forced-colors: active (Windows High Contrast Mode).
  • External links: All links that open in a new tab include a visually-hidden “(opens in a new tab)” announcement for screen reader users.
  • Responsive design: Content is accessible and functional at 200% and 400% zoom levels.
  • JSON-LD schema: Structured data is provided to aid assistive technology and AI agents in understanding page content.

Known Limitations

We are not aware of any current WCAG 2.2 AA failures on this site. However, accessibility is an ongoing process. If you encounter a barrier, we want to know about it immediately.

Assistive Technologies Tested

  • VoiceOver on macOS (Safari)
  • VoiceOver on iOS (Safari)
  • Keyboard-only navigation (Tab, Shift+Tab, Enter, Space, Arrow keys)
  • Windows High Contrast Mode (forced-colors)
  • 200% and 400% browser zoom

ADA Compliance

This website is designed to comply with Title III of the Americans with Disabilities Act (ADA), which requires places of public accommodation to be accessible to people with disabilities. Our WCAG 2.2 AA conformance represents a good-faith effort to meet ADA digital accessibility requirements.

Hope Sessions is an initiative founded on the principle of removing barriers. We take accessibility seriously and will respond promptly to any identified issues.
