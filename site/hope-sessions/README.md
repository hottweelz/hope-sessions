# Hope Sessions Static Page

This folder contains the first static HTML/CSS scaffold for the Hope Sessions public page.

## Current State

- semantic HTML5 page structure
- responsive CSS theme and layout
- accessible navigation and section hierarchy
- placeholder blocks for PMA logo and Hope album cover
- CTA links for partner and artist outreach

## Assets To Replace

2. Add the Hope cover file as `assets/img/hope.png`.
3. Replace the proof placeholders with approved tracks, photos, quotes, and credits.
